# Smart Diet & Fitness Recommendation System 

A web-based application designed to provide users with personalized diet and fitness plans, track their progress, and send timely health reminders. 

This project is built as a **Final Year Project (BS Computer Science, 2022–2026)** at Govt. Postgraduate Jahanzeb College, Swat.

---

## 🛠 Tech Stack

This project is built keeping the architecture simple, modular, and lightweight. **No external frameworks** are used.

*   **Frontend:** HTML5, CSS3, Vanilla JavaScript
*   **Backend:** Core PHP
*   **Database:** MySQL
*   **Security:** Passwords hashed using `bcrypt`

---

## ✨ Core Features

*   **User Management:** Secure registration, login, and comprehensive profile setup (BMI, goals, activity level).
*   **Personalized Recommendations:** A rule-based engine that generates daily diet and workout plans tailored to the user's specific health parameters and dietary habits.
*   **Progress Tracking Dashboard:** Visual analytics for weight trends, macronutrient distribution, daily calorie balance, and weekly activity.
*   **Notification System:** Built-in alerts for meal timings, hydration reminders, and workout schedules.
*   **Motivational Features:** Milestone badges, adherence scores, and daily health tips to keep users engaged.

## 🚀 Getting Started (Local Setup)

To run this project locally, you will need a local server environment like **XAMPP**, **WAMP**, or **MAMP**.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/UbaidUrRehmaanDar/smart-diet-fitness.git
    ```
2.  **Move to local server:**
    Move the `smart-diet-fitness` folder into your local server's web directory (e.g., `C:\xampp\htdocs\` for XAMPP).
3.  **Database Setup:**
    *   Open `phpMyAdmin` (usually at `http://localhost/phpmyadmin`).
    *   Create a new database named `smart_diet_db`.
    *   *(Note: SQL import file will be provided as backend development progresses).*
4.  **Run the application:**
    Open your web browser and go to: `http://localhost/smart-diet-fitness/`
